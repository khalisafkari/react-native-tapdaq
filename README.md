# react-native-tapdaq-ad

tapdaq ads

## Installation

```sh
npm install react-native-tapdaq-ad
```
#### or use yarn

```sh
yarn add react-native-tapdaq-ad
```


## Usage

```js
import Tapdaq from "react-native-tapdaq-ad";
```
